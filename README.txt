
Tarea 2-3 : Algoritmos y complejidad
=============================================
--------------------------------------------------------------------
AMBIENTE DE EJECUCIÓN Y DESARROLLO 
--------------------------------------------------------------------
- Ejecución: Maquina virtual (Ubuntu 18.04.06 LTS)
- Desarollo: Visual Studio Code

--------------------------------------------------------------------
EJECUCIÓN
--------------------------------------------------------------------
- Para compilar fuerza bruta se debe ejecutar el comando:
    g++ -o out fuerzabruta/distanciadeedicion.cpp -Wall

- Para compilar programación dinámica se debe ejecutar el comando:
    g++ -o out prg_dinamica/distanciadeedicion.cpp -Wall

- Para correr el programa se debe ejecutar:
    .\out

--------------------------------------------------------------------
CONSIDERACIONES AL MOMENTO DE EJECUTAR
--------------------------------------------------------------------
- La entrada se ingresa por entrada.txt donde el primer string será
  la palabra y el segundo el objetivo

- Los costos utilizados están en la carpeta costos, estos pueden ser
  generados aleatoriamente con cost_generator, o manualmente

- Si se quiere generar entradas aleatorias también se facilita 
  entrada_generator.py que recibe la palabra objetivo y cambiara el
  archivo de texto dependiendo de los caracteres que use
